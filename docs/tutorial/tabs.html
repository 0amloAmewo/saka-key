Tabs

Tabs

This tutorial uses the default profile's keybindings.

Saka Key lets you, create, duplicate, close, restore, refresh, switch, move, mute, and pin tabs.

Bindings

Create and Duplicate Tabs

  • New Tab - n
  • Duplicate Tab - b

Close and Restore Tabs

  • Close Tab - xx
  • Restore Tab - shift+x
  • Close Other Tabs - xshift+x
  • Close Left Tabs - x[
  • Close Right Tabs - x]

Refresh Tabs

  • Refresh Tab - rr
  • Refresh All Tabs - rshift+r

Switch Tabs

  • Next Tab - q
  • Previous Tab - w
  • First Tab - shift+q or 1
  • Last Tab - shift+w or 0

Move Tabs

  • Move Tab Left - i
  • Move Tab Right - o
  • Move Tab First - shift+i
  • Move Tab Last - shift+o

Mute Tabs

  • Mute/Unmute Tab - m
  • Mute/Unmute All Tabs - shift+m

Pin Tabs

  • Pin/Unpin Tab - shift+p
